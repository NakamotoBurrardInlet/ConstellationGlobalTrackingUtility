## 🇬🇧 English (EN)

### 🚀 Stop Orbiting Constellation Crashes
Welcome to the Aether-Link Collision Avoidance System. As the number of active satellites and stellar constellations around Earth's atmosphere grows, the risk of catastrophic kinetic impacts increases. This system is designed to mathematically devour orbital vectors, track up to 50,000 orbiting machines, and predict potential conjunctions (crashes) before they happen. 

By calculating the precise latitude, longitude, altitude, and velocity of machines from Starlink, GLONASS, NASA, and NORAD, this software provides a real-time, interactive flow of data to safeguard our orbital highways.

### 🧩 Core Components
* **Data Harvester:** Systematically downloads live TLE (Two-Line Element) payload data.
* **Orbital Mechanics Engine:** Calculates real-time physical positioning and trajectory vectors.
* **Collision Predictor:** Scans for overlapping orbital paths and proximity alerts.
* **Tactical Display:** A lightweight, interactive 3D spherical map of the Earth and its surrounding machines.

### ⚙️ How to Download and Install Python 3
To run this system, you need the Python programming language installed on your computer.
1. Visit the official website: [python.org/downloads](https://www.python.org/downloads/)
2. Download the latest version of Python 3 for your operating system (Windows, macOS, or Linux).
3. **Important for Windows:** During installation, ensure you check the box that says **"Add Python to PATH"** before clicking install.

### 📦 Installing Required Packages
Once Python is installed, you must install the software's dependencies using `pip` (Python's package installer). Open your computer's Terminal or Command Prompt and run the following command:
```bash
pip install numpy pandas requests skyfield dash plotly

💻 How to Run the File

    Save the main program script as aether_link.py.

    Open your Terminal or Command Prompt.

    Navigate to the folder where you saved the file using the cd command (e.g., cd Documents/OrbitalProject).

    Execute the system by running:

Bash

python mainapp.py

    Open your web browser and navigate to http://127.0.0.1:8050 to view the live orbital display.
